藍頭機器人隱私政策

版本：1.9 | 最後更新日期：2026年1月17日

本隱私政策規範了用戶在使用藍頭機器人（以下簡稱「本服務」）時的數據處理方式及相關權益。本服務由王曉明（以下簡稱「開發者」）開發並運營。您使用本服務即表示同意本政策的規定及相關法律規範。如您不同意本政策中的任一條款，請立即停止使用本服務。

1. 資料收集

本服務會在以下情況處理資料：

  • 即時指令運行（成員ID、伺服器ID、成員活動狀態）。
  • 回覆使用者（訊息刪除、修改或回覆）。

此外本服務已獲得 特權網關意圖 Privileged Gateway Intents 的完整存取權限，詳細內容為下列：

  • 存在意圖 Presence Intent
  • 伺服器成員意圖 Server Members Intent
  • 訊息內容意圖 Message Content Intent

聲明：本服務不會永久儲存任何訊息內容或個人資料。

2. 資料用途

  • 收集的資料僅用於：
  • 提供即時指令服務。
  • 傳送官方訊息給伺服器擁有者（通常用於緊急狀況）。
  • 確保本服務的功能正常運行。

3. 資料分享

開發者不會與任何第三方分享您的資料，所有數據均僅在本服務內部處理。

4. 資料保護

我們採用現代化技術保護即時處理中的資訊，防止未經授權的存取。

5. 使用者權利

  • 您可隨時移除本服務以停止任何數據處理。
  • 對於功能或隱私的任何疑問，請聯繫開發者獲取協助。

6. 政策修改

本政策可能根據運行需求進行調整，請隨時查看此頁面確認政策是否有更動。

7. 聯絡方式

  • 如對本條款或服務有任何疑問，請聯繫：
  • 開發者：王曉明（dreak.0708）
  • 聯絡方式：請透過 開發者信箱 或至 官方支援伺服器 反應。
